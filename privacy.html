Privacy Policy

Last updated: March 18, 2026

This privacy policy applies to the Stories Club Editor custom GPT and the Recursive Kids Stories Club website hosted at playfulprocess.github.io/recursive-kids-stories-club.

What We Collect

We do not collect, store, or process any personal data.

  • The website is a static site hosted on GitHub Pages. It does not use cookies, analytics, tracking pixels, or any third-party scripts.
  • The custom GPT operates through OpenAI's ChatGPT platform. We do not receive, store, or have access to your conversations with the GPT.
  • If you use the book's edit mode with a GitHub token, that token is stored only in your browser's localStorage and is never sent to us or any server other than GitHub's API.

GitHub API Usage

The custom GPT interacts with the GitHub API to read and write files in the recursive-kids-stories-club repository (or your own fork). This requires a GitHub token configured in the GPT's action settings.

  • The GPT reads book text, illustrations, and configuration files from the repository.
  • When generating illustrations, the GPT writes image files and CSV updates to a preview branch.
  • If you use your own fork, all writes go to your repository only.

AI-Generated Content

  • Illustrations generated by DALL-E through the GPT are subject to OpenAI's usage policies.
  • AI-generated images are labeled as such in the illustrations CSV with the attribution "AI-generated (DALL-E)".

Children's Privacy

The books and website are designed for children, but the GPT and website do not collect any information from children or any other users. There are no accounts, forms, or data submission mechanisms on the website.

The voice chat and page explanation features operate entirely within OpenAI's ChatGPT platform, governed by OpenAI's privacy policy.

Third-Party Services

The project relies on:

  • GitHub / GitHub Pages — hosts the code and website (GitHub Privacy Statement)
  • OpenAI / ChatGPT — powers the custom GPT (OpenAI Privacy Policy)
  • LibriVox — public domain audiobook source (no data collection)

We do not control the data practices of these third-party services.

Open Source

This entire project is open source. You can inspect all code, prompts, and configurations at github.com/PlayfulProcess/recursive-kids-stories-club.

Contact

For questions about this privacy policy, open an issue on the GitHub repository.
