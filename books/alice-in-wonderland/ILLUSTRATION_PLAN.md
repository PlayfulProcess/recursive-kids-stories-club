# Alice in Wonderland — Symbolic Illustration Plan

## 🧌 Core Concept

This is NOT a literal illustration project.

This is a symbolic, artistic reinterpretation of Alice in Wonderland using consistent visual systems.

The goal is to create a second layer of meaning across the book.

## 👱 3. Visual Systems

### 1. Cheshire Cat Continuum

All cats are manifestations of one entity.

Evolution:
- Shadow grin (lyer 1)
- Floating glow (layer 2)
- Full cat (layer 3)
- Disembodied head (layer 4)
- Abstract symbol (layer 5)

Rules:
- Consistent geometry (eyes + grin)
- Consistent color palette
- Increasing abstraction

### 2. Tarot Court System

Wonderland is reinterpreted as a Tarot deck.

Roles:
- Alice = The Fool
- Queen = Dominant archetype
- King = Passive authority
- Cards = Minor Arcana

Style:
- Flat, symmetric compositions
- Gold linework
- Limited palettes

### 3. Nested Stories System

Stories within the story have a distinct visual style.

Style options:
- Sepia ink / parchment
- Woodcut engraving
- Childlike drawings

## 🚀 Production Pipeline

### Phase 1: Scene Extraction
- Parse grammar.json

### Phase 2: Style Definition
